Definitions and Concepts

Efficient management of urban transportation systems, such as parking and EV charging stations, is essential to reduce congestion and improve accessibility. Smart city solutions leverage technology to optimize resources, such as parking availability and real-time charging station management, ensuring sustainable urban mobility.

Identified Business Cases

1. Smart Parking Systems

Problem: Traffic congestion due to inefficient parking space management and lack of real-time availability information.

Solution: Smart parking systems that utilize technologies such as tries and queues to optimize parking availability and direct vehicles to open spots in real-time.

Algorithmic Applications:

Parking Slot Availability using Trie

Functionality: A trie can store and retrieve available parking slots based on location and type (e.g., compact, regular, or handicapped) efficiently.

Example Usage:

  • Each node in the trie represents a level of detail (e.g., city > parking lot > slot number).
  • Quick lookup for the nearest available parking slot in a specific area.

#include 
using namespace std;

// TrieNode class to represent each level of the Trie
class TrieNode {
public:
    unordered_map children; // Map to hold children nodes
    bool isEndOfSlot; // Flag to mark end of a valid slot
    
    TrieNode() : isEndOfSlot(false) {}
};

// ParkingTrie class to manage parking slots using Trie
class ParkingTrie {
    TrieNode* root; // Root node of the trie

public:
    ParkingTrie() {
        root = new TrieNode();
    }

    // Insert a parking slot in the Trie
    void insert(const string& slot) {
        TrieNode* node = root;
        for (char ch : slot) {
            if (node->children.find(ch) == node->children.end()) {
                node->children[ch] = new TrieNode();
            }
            node = node->children[ch];
        }
        node->isEndOfSlot = true;
    }

    // Search for a parking slot in the Trie
    bool search(const string& slot) const {
        TrieNode* node = root;
        for (char ch : slot) {
            if (node->children.find(ch) == node->children.end()) {
                return false;
            }
            node = node->children[ch];
        }
        return node->isEndOfSlot;
    }
};

int main() {
    ParkingTrie trie;
    trie.insert("City1_LotA_Slot3");
    trie.insert("City1_LotB_Slot1");
    
    // Checking availability of slots
    cout << "Is Slot City1_LotA_Slot3 available? " << (trie.search("City1_LotA_Slot3") ? "Yes" : "No") << endl;
    cout << "Is Slot City1_LotA_Slot4 available? " << (trie.search("City1_LotA_Slot4") ? "Yes" : "No") << endl;
    
    return 0;
}
Dynamic Queue for Real-time Slot Allocation

Functionality: A queue can dynamically manage the order of vehicles waiting for parking slots.

Example Usage:

  • Vehicles are queued as they approach the parking area.
  • As slots become available, the queue ensures that the nearest or most eligible vehicle is allocated a spot based on priority (e.g., arrival time or special needs).
#include

#include 
using namespace std;

// Structure to hold information about vehicles
struct Vehicle {
    string licensePlate;
    int arrivalTime;
    
    // Custom comparison for priority queue (longest waiting vehicle first)
    bool operator>(const Vehicle& v) const {
        return arrivalTime > v.arrivalTime;
    }
};

// Class to manage parking queue using priority queue
class ParkingQueue {
    priority_queue, greater> queue; // Priority queue to manage vehicles
    
public:
    // Add vehicle to the queue
    void addVehicle(const string& licensePlate, int arrivalTime) {
        queue.push({licensePlate, arrivalTime});
    }

    // Allocate slot to the vehicle at the front of the queue
    void allocateSlot() {
        if (!queue.empty()) {
            Vehicle v = queue.top();
            queue.pop();
            cout << "Allocating slot to vehicle: " << v.licensePlate << endl;
        } else {
            cout << "No vehicles in queue!" << endl;
        }
    }
};

int main() {
    ParkingQueue pq;
    pq.addVehicle("AB123CD", 2);
    pq.addVehicle("XY987ZT", 1);
    
    // Allocating slots based on priority (arrival time)
    pq.allocateSlot(); // Should allocate to XY987ZT
    pq.allocateSlot(); // Should allocate to AB123CD
    
    return 0;
}
